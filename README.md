# laboro-email-assets

Obrázky pro **e-mailový podpis** Laboro ateliér, s.r.o. — hostované na veřejné URL,
aby je e-mailoví klienti (Gmail, Outlook, Apple Mail) spolehlivě zobrazili.

## Soubory a `raw` URL

| Soubor | Použití | URL |
|---|---|---|
| `logo-laboroatelier-email.png` | logo (na světlé pozadí) | `https://raw.githubusercontent.com/kadrmas126-cpu/laboro-email-assets/main/logo-laboroatelier-email.png` |
| `logo-laboroatelier-email-white.png` | logo (na tmavé pozadí) | `https://raw.githubusercontent.com/kadrmas126-cpu/laboro-email-assets/main/logo-laboroatelier-email-white.png` |
| `ikona-adresa.png` | ikonka adresy | `.../ikona-adresa.png` |
| `ikona-telefon.png` | ikonka telefonu | `.../ikona-telefon.png` |
| `ikona-email.png` | ikonka e-mailu | `.../ikona-email.png` |
| `ikona-web.png` | ikonka webu | `.../ikona-web.png` |

Logo: 360×131 px (retina, zobrazuje se na 180×65). Ikonky: 30×30 px (retina, zobrazují se na 15×15). Vše PNG s průhledným pozadím, firemní zelená `#A7C139`.

> Repo je veřejné schválně — e-mailoví klienti musí obrázky stáhnout bez přihlášení.
